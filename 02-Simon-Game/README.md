# 🎮 Simon Says Game

A minimal and interactive **Simon Says** memory game built using **HTML, CSS, and JavaScript**.  
The project emphasizes clean UI, clear logic, and strong front-end fundamentals.

---

## ✨ Introduction

Simon Says is a classic memory-based game where the system generates a sequence of colors, and the player must reproduce the sequence correctly.  
With every successful round, the sequence grows longer, gradually increasing the difficulty.

This project demonstrates how simple technologies can be combined to create an engaging and structured game experience.

---

## 🌟 Key Features

- Start the game using keyboard input  
- Visual feedback for both system and user actions  
- Progressive level-based difficulty  
- Automatic game-over detection with score display  
- Clean and minimal user interface  

---

## 🛠️ Built With

- **HTML5** — semantic structure  
- **CSS3** — layout, colors, and visual feedback  
- **JavaScript (ES6)** — game logic and interactions  

No frameworks or external libraries are used.
